#### HashChain 이슈
- 패킷 블록 구성
  - DATA 영역에 next hash 가 포함되도록 함
    - 첫번째 패킷은 RSA sign
  - 두번째 패킷부터 sign 영역에 next hash 활용
    - DATA 영역에 next hash 포함
    - 동일한 next hash 가 sign 에 활용

```
# 구성 예)

     +-   +----------+     +----------+     +----------+ 
     |    |   DATA   |  +->|   DATA   |  +->|   DATA   |
data |    |----------|  |  |----------|  |  |----------|
     |    |next hash |--+  |next hash |--+  |  0 hash  |
     +-   +----------+     +----------+     +----------+
sign |    |   RSA    |     |next hash |     |  0 hash  |
     +-   +----------+     +----------+     +----------+
```
